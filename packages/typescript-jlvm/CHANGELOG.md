# Changelog

All notable changes to `@constellation-network/metagraph-sdk-jlvm` (the metakit
JSON-Logic VM) will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

Byte-for-byte parity with the metakit (Scala) reference is verified on every CI
run against the shared conformance vectors in `/shared`.

## [1.8.0-rc.7] - 2026-07-01

Aligned byte-for-byte with metakit `io.constellationnetwork:metakit_2.13:1.8.0-rc.7`.

### Added
- `hex_to_int` opcode — unsigned big-endian hex → integer (arbitrary precision),
  gas pinned equal to `%`. Vectors: `json_logic_test_vectors.json` (1.6.0).
- Map `set` and `unset` opcodes — immutable dict write/delete with an evaluated
  key (last-wins on `set`; absent-key `unset` is a no-op), gas equal to `merge`.

- `verifyOrdinalCatalogProof` — the ordinal-catalog attestation verifier
  (`ordinal → committed MPT root` against a trusted catalog root, built on
  `smt_verify`), byte-aligned with the metakit reference. Vectors:
  `ordinal_catalog_test_vectors.json` (1.0.0). See `docs/committed-roots.md`.

### Changed
- Σ-protocol DoS hardening, matching the Scala reference: `SIGMA_MAX_PROOF_NODES`
  = 4096, `SIGMA_MAX_PROOF_DEPTH` = 64, `SIGMA_MAX_MESSAGE_BYTES` = 4096, and
  strict rejection of unknown fields in every proposition/proof node
  (`sigmaRejectUnknownFields`). Vectors: `zk_opcode_test_vectors.json` (1.12.0).
- Gas vectors extended to `1.2.0` with `hex_conversion` (`hex_to_int`) and
  `map_mutation` (`set`/`unset`) categories, generated by the Scala meter.

## [1.8.0-rc.3] - 2026-06

First independent publish of the JLVM package after the TypeScript monolith was
split into three packages.

### Added
- Full base JLVM opcode set (control-flow, logical, comparison, arithmetic,
  array, string, object-map, utility).
- Full ZK / auth-DB opcode suite — `poseidon`, `pmt_verify`, `smt_verify`,
  `mpt_verify`, `mpt_prefix_verify`, `groth16_verify`, `ecvrf_verify`,
  `bn254_add`/`mul`/`pairing`, `bls_verify`/`bls_aggregate_verify`,
  `schnorr_verify`, `prove_dlog_verify`, `prove_dhtuple_verify`, `sigma_verify`
  — reaching full parity with the Rust and Scala JLVMs (all 17 ZK vector
  categories green).
- Static gas metering aligned with the Scala `JsonLogicGasEstimator`.
